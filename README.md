# Tạo trang giới thiệu bản thân trên git
## 1 vài câu lệnh
- git config --global user.name: kiểm tra trên máy đang dùng git gì
- git init: khởi tạo rep
- git remote add origin <link>: nối vào link sau khi khởi tạo
- git add . : thêm tất cả file và thư mục hiện tại vào staging area của Git
- git commit -m "xxx": commit trước khi đẩy lên
- git push -u origin main(master): đẩy file or thư mục lên git
